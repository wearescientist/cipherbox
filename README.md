# Credential Vault 🔐

> 纯前端、零依赖、单 HTML 文件的本地凭据管理器。所有数据在本地浏览器加密存储，不上传任何服务器。

## 快速开始

1. **下载** `index.html`
2. **双击打开**（浏览器需支持 Web Crypto API — 现代 Chrome / Edge / Firefox 均支持）
3. **首次使用** → 自动显示注册页面，设置主密码（至少 8 位）进入
4. **已有数据** → 显示解锁页面，输入密码即可

就是这么简单，无需安装、无需注册、无需服务器。

## 功能

### 支持四种凭据类型

| 类型 | 适用场景 |
|------|----------|
| 账号/密码 | 网站登录、应用账号 |
| 平台/API | 开发用 API Key、Secret Token |
| 链/地址 | 加密货币钱包地址、合约地址 |
| 平台/UID | Discord / Steam / 游戏平台 ID 等 |

### 快捷键一览

| 快捷键 | 功能 |
|--------|------|
| `Ctrl+N` | 新建条目 |
| `Ctrl+S` | 保存当前编辑 |
| `Ctrl+F` | 聚焦搜索框 |
| `Ctrl+E` | 选中条目 → 载入编辑器 |
| `Ctrl+D` | 删除选中条目（需二次确认） |
| `Ctrl+L` | 立即锁定保险库 |
| `Ctrl+Shift+E` | 导出加密备份 |
| `Esc` | 清空编辑器 |
| `↑ / ↓` | 列表上下选择 |
| `1 / 2 / 3 / 4` | 切换四种类型标签 |
| `Enter` | 登录页面快速提交 |

### 安全设计

- **主密码** — 至少 8 位，通过 PBKDF2 派生加密密钥
- **加密算法** — AES-256-GCM，加密同时保证数据完整性
- **敏感值隐藏** — 默认显示为圆点，点击可查看（15 秒后自动重新隐藏）
- **剪贴板自动清理** — 复制后 30 秒自动清除；锁定保险库时立即清除
- **删除二次确认** — 防止误删
- **重复检测** — 保存时自动检查是否已存在相同条目
- **搜索框一键清空** — 输入内容后出现 ✕ 按钮，或按 `Escape` 清空

## 数据存储与备份

- 所有数据在 `localStorage` 中以 **AES-256-GCM 加密**后存储
- 定期导出备份（`Ctrl+Shift+E`），保存为 `.json` 文件
- 支持重新导入备份文件
- ⚠️ 清除浏览器数据 = 丢失数据，请养成备份习惯

## 批量导入

切换到「批量导入」模式，支持一行一条，使用 `TAB`、`|`、`----` 或第一个逗号作为分隔符。

```
Binance    api_xxx
ETH        0x123...
X | 123456789
```

## 适用人群

- 不想注册第三方密码管理服务的用户
- 开发者需要一个本地存放 API Key 的工具
- 追求极简、离线、单文件方案的人

## 安全提示

> 本工具的数据完全存储在浏览器 `localStorage` 中。安全性取决于你的主密码强度和设备物理安全。建议使用强密码（12 位以上混合字符）并为电脑设置锁屏。
